# Bayar Pake Apa??

Sebagai ecommers terbaik saat ini. Tentunya TotoKoko ingin memfasilitasi metode pembayaran pengguna-penggunanya. Atasan **Dumdum**ğŸ£ meminta untuk menambahkan metode pembayaran Emoney pada Payment Gateway TotoKoko. Akan tetapi code yang **Dumdum**ğŸ£ buat tidak _extensible_ (sulit diperluas). Sehingga metode pembayaran baru sulit untuk ditambahkan dan dipelihara.

Codingan yang dibuat **Dumdum**ğŸ£ juga kurang tepat menurut sang atasan karena tidak menerapkan prinsip-prinsip pengembangan perangkat lunak yang telah diajarkan.

## Soal

[1] Analisis **kekurangan-kekurangan** dari code yang telah dibuat oleh **Dumdum**ğŸ£

[2] Bantulah **Dumdum**ğŸ£ untuk menemukan **design pattern** yang sesuai dengan masalah tersebut. **Jelaskan alasan pemilihan design pattern**

[3] Untuk mempermudah **Dumdum**ğŸ£ memahami codinganmu sertakan **class diagram** beserta penjelasannya.

[4] Silakan **kembangkan** code yang telah dibuat DumdumğŸ£ dengan menerapkan **design pattern** yang dipilih beserta **prinsip-prinsip perangkat lunak yang telah dipelajari (codingan)**

[challange] Tambahkan metode pembayaran Emoney pada code baru yang dibuat.
